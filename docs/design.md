## 数据库技术及应用大作业——T大体育公园信息管理系统

本应用模拟了一个体育场馆中信息的管理过程，包括场地的预约申请审批，器材的租借等等。

### 1.需求分析

使用该系统的的有三类人：管理员，体育老师，学生。我们把它们抽象为用户。

所有用户都需要的功能有：

* 登录 b 

* 查看场地信息和占用情况 b

* 查看器材信息 b

* 查看各种运动的情况//

* 查看各个院系的运动情况//

* 查看各个班级的运动情况//

#### 管理员

管理员需要实现的功能：

* 查看所有场地预约 b

* 查看所有未审批的场地预约，并审批 b

* 记录器材租借 b

* 设置场地信息 

* 设置物品信息

#### 老师

老师需要实现的功能：

* 提交场地预约申请 b

* 查看个人历史预约

* 查看自己院系运动情况//

#### 学生

学生需要实现的功能：

- 提交场地预约申请（特长生不用审批）b

- 查看个人历史预约

- 查看自己班级运动情况//

- 查看自己院系运动情况//
